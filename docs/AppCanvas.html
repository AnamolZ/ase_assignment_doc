Class AppCanvas
The AppCanvas class provides a simple drawing canvas for creating shapes, lines, and text. It allows for customizable pen color and size, and supports basic drawing operations like circles, rectangles, and triangles.
Syntax
C#
    public class AppCanvas : ICanvas

Documentation Preview
Constructor AppCanvas
Creates a new canvas with customizable size and pen thickness.
Syntax
C#
    public AppCanvas(int xSize = 421, int ySize = 314, int penSize = 5)
Parameters
xSize
Width of the canvas in pixels. Default is 421.
ySize
Height of the canvas in pixels. Default is 314.
penSize
Thickness of the pen used for drawing. Default is 5.
